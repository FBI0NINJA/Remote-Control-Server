# 🎮 Remote Control Server

![Private Project](https://img.shields.io/badge/Project-Private-red) ![Safe for LAN](https://img.shields.io/badge/Safe-for%20LAN-blue) ![Node.js](https://img.shields.io/badge/Node.js-v18-green)

A personal project to **remotely control your PC** applications like Spotify, PUBG (Gameloop), LDPlayer, and system actions (Sleep, Restart, Shutdown) via a **web interface**.

---

## ⚡ Features

- 🎧 Open **Spotify** directly  
- 🎮 Start **PUBG (Gameloop)** instantly  
- 📱 Launch **LDPlayer** Android emulator  
- 💤 Put PC to **Sleep**  
- 🔄 **Restart** your PC  
- ⛔ **Shutdown** your PC  
- 🔒 **Password-protected access**  

> **Note:** The source code is **private**. If you want the code or setup instructions, please contact the developer directly.
